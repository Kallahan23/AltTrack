# AltTrack
Cross-Platform Ionic App for tracking crypto currency prices.
Defaults to AUD.

## Currently Supported Coins
Using BTC Markets API:
- Bitcoin (BTC)
- Bitcoin Cash (BCH)
- Ethereum (ETH)
- Ethereum Classic (ETC)
- Litecoin (LTC)
- Ripple (XRP)

Using Coinspot API
- Dogecoin (DOGE)

Using CoinMarketCap API
- Miota (IOTA)
